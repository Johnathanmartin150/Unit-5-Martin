Text Formatting

There are three main types of text formatting. Those are bold, italics. and underline.

InLine vs. Block Element

A block element moves tge content to a new Line. The paragraph and header elements are block elements. An inline element does not display its content on a new Line. It keeps the content on the same line as the previous content. Bold italics and underLine are all inLine elements

Images

An image element is an inline element which inserts pictures into the document. An image element is self-closing, which means it does not have a closing tag. It has one major attribute, src, which is filepath that points to the images' location. those filepaths can either be absolute or relative.

Simple image Example

Image Attributes

there are 4 attributes commonly used for images. width and height are used to resize images. alt allows to use text descriptions of images for accessibility and broken images. titles allows a hover-over text to appear.

Width and Height

the width and height can be measured in many different ways but the more common one are pixels and percentages. Using pixels, you set the specific, precise size based on pixels using percentages you set the dimensions based on percentage of the entire page. in either case, setting the width alone respects aspect ratio, while setting both does not.

Alt Attribute

EVERY picture should at LEAST have the alt attribute. The alt attribute helps with the screen readers for visually imparied in the cases that the image is broken.
