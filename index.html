Text Formatting

There are three main types of text formatting. Those are bold, italics. and underline.

InLine vs. Block Element

A block element moves tge content to a new Line. The paragraph and header elements are block elements. An inline element does not display its content on a new Line. It keeps the content on the same line as the previous content. Bold italics and underLine are all inLine elements

Vertical Text Spacing

There are two elements that help with spacing between block elements. ALine break element adds an empty line and moves the content down to a new line. Its similar to the ENTER key in google/word doc. A horizontal rule is a line break with a horizontal bar across the page. Both of these are block elements.

Attributes

Attributes are properties which can change an HTML element. Attributes are written in the opening tag of ANY html element. The most common attributes are src, href, class, id, and style.
